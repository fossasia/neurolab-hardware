# metadata-viewer

Application takes in multiple files and displays the name, size, type and last modified date of them. 

##To Use:
1. Clone Repository  
2. Navigate to index.html  
3. Upload a File  
4. View Metadata  
OR  
1. Go to http://codethejason.github.io/metadata-viewer
